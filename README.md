# Project Bundles

This repository consists of a collection of mini web development projects built while learning foundational frontend technologies. Each project focuses on mastering specific concepts, ranging from DOM manipulation and state management to asynchronous API handling.

## 🚀 Project Overview

| Folder Name | Description | Tech/Concepts Learned |
| :--- | :--- | :--- |
| **`Book_library`** & **`bookLibrary2`** | A classic data management app to add, remove, and track a collection of books. | DOM Manipulation, State Tracking |
| **`Calendar`** | A dynamic calendar view handling date logic, month transitions, and layout grids. | Date Objects, Grid UI Layouts |
| **`bmi`** | A calculator that takes numerical user input, computes BMI, and displays conditional health ranges. | Input Handling, Form Validation |
| **`calculator`** & **`calculator2`** | Essential fundamental logic engines built to process math operations and live UI updates. | Event Handlers, Operator Logic |
| **`dropdowncolor`** | A neat UI utility that dynamically changes page themes/colors based on a dropdown selection. | Change Events, Dynamic Styling |
| **`fetch recipe`** | An application connecting to an external API to fetch and dynamically render recipe details. | Async/Await, Fetch API, JSON Parsing |

---

## 🛠️ Core Tech Stack Covered
* **HTML5** – Structured semantic layouts.
* **CSS3** – Responsive design, grid systems, and dynamic UI styling.
* **JavaScript (ES6+)** – Asynchronous programming, event-driven architecture, and DOM manipulation.
